# RoofCheck - Build Log

## v3.0 - Lead Gen Pivot (2026-01-29)

**Major Changes:**
- Pivoted from B2B SaaS (inspectors) to B2C lead gen (homeowners → roofers)
- Renamed to **RoofCheck** (consumer-friendly)
- Added roofer-facing page for network signups
- Created full business plan

**New Files:**
- `index.html` - Consumer-facing tool (upload → analysis → lead capture)
- `for-roofers.html` - Roofer network signup page
- `BUSINESS-PLAN.md` - Full strategy document

**The Funnel:**
1. Traffic (Google/FB/TikTok) → Consumer uploads roof photo
2. AI analyzes age/condition → Returns recommendation
3. If roof needs work → "Get 3 Free Quotes" lead form
4. Lead sold to local roofers ($65 shared, $149 exclusive)

**Unit Economics:**
- Traffic CPA: $15-25
- Lead capture rate: 20-30%
- Revenue per lead: $65-149
- Margin: 30-65%

**Key Features:**
- 4 recommendation levels: Good / Monitor / Attention / Urgent
- Lead form only shows when roof needs work (no spam)
- Full legal disclaimers included
- Mobile-responsive
